# VersusLog  

## About
汎用のTCG戦績管理ツールです。  
下記機能を備えています。  
・ログ入力、表示機能  
・デッキごとの戦績表示機能  
・トップメタデッキ分析(1週間、1か月)

## Supported language
日本語のみ(Japanese Only)
